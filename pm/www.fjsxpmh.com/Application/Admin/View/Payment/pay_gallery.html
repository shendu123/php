BeeCloud支付配置
支付简介: 本系统集成了BeeCloud支付云服务平台，该平台目前整合了以下支付平台。请注册BeeCloud后续设置相应支付接口。
BeeCloud比可科技有限公司（北京，苏州，南京），是国内首家提供一站式移动支付解决方案的云服务平台，提供安全稳定的分布式云后端服务，为应用扫清支付障碍，保障用户支付过程安全流畅。
APP ID : beecloud平台的APP ID
APP Secret: beecloud平台的APP SECRET
MASTER SECRET: beecloud平台的MASTER SECRET
TEST SECRET: beecloud平台的TEST SECRET, for sandbox
Webhook URL: {$webhookurl}(复制该地址到BeeCloud所添加项目的Webhook URL中，需要在公网内测试。)
支付平台：

  • 支付宝(PC即时到账支付)

  • 支付宝(移动网页支付)

  • 支付宝(扫码支付)

  • 微信(扫码支付)

  • 微信(公众号支付)注：微信浏览器打开

  • 京东(PC网页支付)

  • 京东(移动网页支付)

  • 百度(PC网页支付)

  • 百度(移动网页支付)

  • PayPal支付

  • 易宝(PC网页支付)

  • 易宝(移动网页支付)

  • 易宝(点卡支付)

  • 银联(PC网页收银台)

  • 联通(移动网页支付)

  • BeeCloud网关支付

  • BeeCloud快捷支付

  • BeeCloud微信支付

  • BeeCloud微信移动网页支付

  • BeeCloud微信扫码支付

  • BeeCloud微信刷卡支付

  • BeeCloud支付宝扫码支付

  • BeeCloud支付宝刷卡支付
启用相应接口请确定您已在BeeCloud中设置相应接口，如未在BeeCloud设置相应接口请设置为禁用！
易宝支付: 50位以内数字和/或字母组合，易宝移动网页（一键）支付用户唯一标识符，用于绑定用户一键支付的银行卡信息
疑问和解决: 【微信支付】请到【微信公众平台】->【微信支付】->【开发配置】->【支付授权目录】设置为：{$weixinurl}
【微信公众号支付】请修改网站文件【\ThinkPHP\Library\Vendor\beecloud\WxPay\WxPay.pub.config.php】下的APPID、MCHID、KEY、APPSECRET为您的微信平台对应值
【易宝支付】请到易宝控制台设置通知地址为：https://apidynamic.beecloud.cn/1/pay/callback/yeeWeb
如有疑问请联系客服协助您解决！
