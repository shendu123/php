拍卖订单配置
支付期限： 天  订单生成后，买家的支付期限。如果过期将扣除买家保证金，扣除佣金后自动转入卖家账户。
发货期限： 天  买家确认支付后，卖家发货时间期限。如果过期未发货，将扣除卖家保证金转入卖家账户。
收货期限： 天  卖家发货后，买家确认收货时间。如过期将自动确认收货。
买家评价期限： 天  买家确认收货或自动确认收货后，买家的评价期限。如过期将自动确认默认好评。
卖家评价期限： 天  买家评价后，卖家评价买家的时间。如过期将自动好评。
违约处理： 如未按时支付或未按时发货网站抽取保证金的%，剩余部分转入对方账户
修改后需清理缓存【首页】->【缓存清理】（选择网站配置缓存）进行清理。
*不填写或者0为永久有效！*
*该配置为订单生成后支付时间段，如在该时间段未进行支付，用户将无法进行支付！保证金将被扣除！*
